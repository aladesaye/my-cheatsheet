HTML FUNDAMENTAL ELEMENT REFERENCE

Table Tags

HTML FUNDAMENTAL ELEMENT
S/no Tag Name Description
1 <p></p> paragraph This tag is used for portraying a paragraph in HTML elements.
2 <h></h> heading This tag is used for portraying a heading in HTML elements. it has family from h1 to h6. The more number in the heading tag, the more smaller the words.
3 <div></div> Division This is a tag that divides the page of HTML into division
4 <em></em> Emphasis This tag is used for emphasising a elements in HTML. It renders the text italics
5 <strong></strong> strong This tag highlights important text in HTML elements. It renders a bold emphasis.
6 <ul></ul> unordered list This tag is used for listing elements in HTML elements with bullet points. This is used with the list(li) tag as well.
7 <img/> Image This tag is used for portraying an image in HTML, it is a self closing tag i.e it does not require a closing tag. It has other attribute which include the following:

Image Attributes

Image Attribute
S/no Tag Name Description
1 src Source This is the source of the image being used, the value can be the url of the image from another websites or on your local address in the project folder.
2 alt Alternative This is the alternative text which gives the image more emphasis and more description.
